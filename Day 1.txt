10 Git Commands
.............................

1.git init - To initialize new local repository

2.git commit -m "commit msg"  - Record the staged changes to the repository with a descriptive message

3.git --version  - Check installed git version

4.git add ./git add<file> - Stages Changes in specified files or all changes in the current directory for the next commit

5.git config --global user.name "name" - sets username

6.git config --global user.email "email" -sets email

7.git config --list - view all git configuration settings 

8.git clone <repository URL> -clone/copy a repository from GitHub

9.git remote -v - view connected remote repositories

10.git status -show modified and untracked files

11.git reset<file name> -  to remove file

12.git log -   view commit history

13.git branch --all - list all branches

14.git branch <branch name> - create a new branch

15.git checkout -b <branch-name> - create and switch to a new branch

16.git push

17.git pull

18.git fetch
